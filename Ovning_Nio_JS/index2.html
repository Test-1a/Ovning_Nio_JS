Bish-Bosh 2.0

To start the game:

First, state how many cycles to run the game, a number between 1 and 999. Then, state which should be the first number, a number between 1 and 9 (prints out Bish). After that, state the second number, a number between 1 and 9 (prints out Bosh). Finally, click on the button labeled "Start" to start the game.
